Identyfikatory (ID) w CSS

Pojęcie Znaczenie pojęcia
Co znaczy skrót ID? ID to skrót od "Identifier" - unikalny identyfikator elementu w dokumencie HTML
Jaki jest cel stosowania ID? Pozwala na przypisanie unikalnego stylu lub zachowania do konkretnego elementu strony
Jakie jest znaczenie znaku # W CSS znak # poprzedza nazwę identyfikatora podczas jego definiowania
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W HTML: <element id="nazwa">, w CSS: #nazwa { właściwości }
Uwagi na temat stosowania ID (trzy konieczne wyliczenia)
  1. ID musi być unikalne w całym dokumencie
  2. Element może mieć tylko jedno ID
  3. ID ma wyższy priorytet niż klasy w kaskadowaniu stylów
